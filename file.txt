TRACCIA -

Modellizzare la struttura di una tabella per memorizzare tutti i dati riguardanti delle auto usate messe in vendita da un concessionario.

TABELLA - 

AUTOMOBILE

id - INT - NOT_NULL, UNIQUE, AUTO_INCREMENT - PRIMARY KEY
marca - VARCHAR(25) - NOT_NULL 
modello - VARCHAR(25) - NOT_NULL
nuova - BOOL (TINYINT (1)) - NOT_NULL DEFAULT(0)
usata - BOOL (TINYINT (1)) - NOT_NULL DEFAULT(0)
km - INT - NULL
colore - VARCHAR(25) - NULL
prezzo - INT - NOT_NULL 
anno - YEAR - NULL
alimentazione - VARCHAR (25) - NOT_NULL 
airbag - BOOL (TINYINT (1)) - NOT_NULL DEFAULT(0)
cambio manuale - BOOL (TINYINT (1)) - NOT_NULL DEFAULT(0)
cambio automatico - BOOL (TINYINT (1)) - NOT_NULL DEFAULT(0)
cambio semiautomatico - BOOL (TINYINT (1)) - NOT_NULL DEFAULT(0)
marce - TINYINT - NOT_NULL 
proprietary - TINYINT - NULL
